# 🚗 Hindustan Diesel Garage

A modern and responsive web application for a diesel garage business, built using React and Vite. This platform showcases auto parts, services, and provides a clean user interface for customers.

---

## 🔥 Features

* 🛠️ Auto Parts Listing
* 🚘 Garage Services Information
* 🖼️ Gallery Section
* 🔐 User Authentication (Login/Register)
* 📱 Fully Responsive Design
* ⚡ Fast Performance with Vite

---

## 🧰 Tech Stack

* ⚛️ React.js
* ⚡ Vite
* 🎨 CSS
* 🔥 Firebase (Authentication / Backend)
